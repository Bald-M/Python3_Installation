## Red Hat/CentOS
## Install
~~~
[root@localhost ~]# unzip Python3_Installation.zip
[root@localhost ~]# cd Python3_Installation/
[root@localhost Python3_Installation]# sh install.sh
~~~
## Uninstall
~~~
[root@localhost ~]# unzip Python3_Installation.zip
[root@localhost ~]# cd Python3_Installation/
[root@localhost Python3_Installation]# sh uninstall.sh
~~~
